Quick and easy integration of publishers' codes from LoadAds.Com for WordPress websites monetization



== Description ==

LoadAds proudly presents a proprietary ad serving technology packed as an ad network that manages large volumes of impressions on a daily basis.
The "LoadAds Publisher Code Integration" plugin offers the possibility to easy integrate the LoadAds.Com publishers codes, with the option to temporarily disable the ads delivery without having to uninstall or deactivate the plugin.

More details here: http://loadads.com/get/p-wordpress_plugin



== Installation ==

1. Install the plugin through the WordPress plugins page directly or upload the plugin files to the "/wp-content/plugins/" directory.
2. Activate the plugin through the 'Plugins' menu screen in WordPress
3. Register as a publisher on LoadAds.com or use the ID/URL string from an existing panel.
4. Configure  the plugin usage for popunder or overlay or redirect activities by pasting the ID string into the filed shown.
5. Use the API key form loadads.com to check the hits flow status (API stats are updated only once every hour)

That's all.
LoadAds inventory will appear on the wordpress pages, generating instant earnings.
